# DP-300: Preguntas de Simulación del Examen

Este directorio contiene preguntas de simulación y recursos para prepararse para el examen **Microsoft Certified: Azure Data Engineer Associate (DP-300)**, enfocado en la administración de bases de datos Azure SQL.

## Formato de Nomenclatura de Archivos

Todos los archivos PDF con preguntas de simulación deben seguir estrictamente el siguiente formato de nomenclatura para mantener la organización:

`DP300-SIM<numero pregunta general>-T<numero del topico>-<numero de pregunta del topico>.pdf`

### Desglose del formato:

*   **`DP300`**: Prefijo estándar del examen.
*   **`SIM<numero pregunta general>`**: Número de simulación general (por ejemplo, `SIM001`, `SIM002`).
*   **`T<numero del topico>`**: Identificador del área temática o dominio del examen (por ejemplo, `T01`, `T02`, `T03`).
*   **`<numero de pregunta del topico>`**: Número de pregunta dentro de ese tema específico (por ejemplo, `001`, `002`).

### Ejemplos de Nombres de Archivos Válidos:

*   `DP300-SIM001-T01-001.pdf` (Simulación 1, Tópico 1, Pregunta 1)
*   `DP300-SIM002-T05-010.pdf` (Simulación 2, Tópico 5, Pregunta 10)
